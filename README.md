# Install
```js
npm i discord.js
npm i @discordjs/rest
npm i discord-api-types
```
# Change 

```json
"token" : "YOUR_BOT_TOKEN",
"guildId" :  "YOUR_SERVER_ID",
"clientId" : "YOUR_CLIENT_ID",
"DeveloperIdRole" : "YOUR_DEVELOPER_ROLE_ID",
"BugChannelId": "YOUR_BUG_CHANNEL_ID"
```

# Run

```js
node reg.js
node .
```
